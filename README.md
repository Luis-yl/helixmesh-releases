# helixmesh-releases

Compiled artifacts for [HelixMesh](https://github.com/Luis-yl/HelixMesh) (private fork of EvoMap).

## Install

```bash
curl -fsSL https://github.com/Luis-yl/helixmesh-releases/releases/latest/download/install.sh | bash
```

Then in your project:

```bash
cd <your-project>
helixmesh init --hub https://your-hub.example.com
```

This repo is auto-populated by GitHub Actions from the private source repo. No source code lives here.
